**A scalable, distributed logistics decision system inspired by large-scale e-commerce platforms. This project simulates how delivery options are processed, optimized, and fulfilled across multiple warehouses under real-world constraints.**

# Overview

The **Distributed Fulfillment Engine** is designed to replicate the core logic behind modern e-commerce fulfillment systems. It focuses on optimizing delivery decisions such as warehouse selection, order routing, and shipment processing while maintaining high performance and scalability.

# Tech Stack

- **Backend** : FastAPI (Python)
- **Database** : PostgreSQL
- **Frontend** : HTML, CSS, JavaScript
- **Architecture** : RESTful APIs, Modular Design
- **Concepts** : Distributed Systems, Optimization Logic, SDLC
